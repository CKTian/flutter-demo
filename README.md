# iwms_demo

基于dart语言 使用flutter框架的移动端app

ios端采用oc语言 android端采用kotlin语言

无后台服务 仅限前端样式demo

目前实现页面功能：

- 根据系统语言环境 展示对应语言 支持切换语言环境（ios）（Android没有测试）

- 登陆/登出/切换组织/修改密码

- 上拉加载 下拉刷新

- 搜索框+列表展示
